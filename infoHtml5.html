HTML5 = Hypertext Markup Language revision 5

(Project Development Code)

HTML is the first step developers need to take for creating web pages. HTML5 is its latest version that provides necessary structure and presentation of World Wide Web contents. Compared to its previous version it supports traditional HTML and XHTML-style syntax and other new features in its markup, New APIs, XHTML and error handling.
Getting hands-on with HTML5 will expose you to building single page applications for browsers. By the end of this module you will be able to build a single page web application.
