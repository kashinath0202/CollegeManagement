ANGULAR

Angular is a structural framework for developing feature rich Single Page Applications (SPA). The need for SPA and limitations of JavaScript (ECMA 5) has made TypeScript (ECMA 6) a super script of JavaScript as a programming language for Angular.
Using TypeScript plus exploring core features of Angular like components, services, event binding, data binding, directives, routing and navigation, authentication and authorization, HTTP services, Form handling will take your front-end knowledge to the next level. By the end of this you will be able to develop and deploy a front end application with Firebase as the backend.
